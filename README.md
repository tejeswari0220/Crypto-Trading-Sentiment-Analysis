# Crypto Trading & Market Sentiment Analysis

This project analyzes the relationship between crypto trading performance (Profit & Loss) and market sentiment (Fear & Greed Index).

## Dataset Overview
- Total Merged Records: 35,864 trades
- Source: Combined trading historical logs with daily Fear & Greed Index data.

## Key Insights From Analysis
- **Extreme Greed:** Highest Win Rate (~55.3%) and generated over $1.15M in PnL.
- **Fear:** Highest overall profit zone with $1.77M Total PnL, though win rate was moderate (~38.1%).
- **Extreme Fear:** Lowest activity and lowest win rate (~29.2%).

## Files Included
- `analysis.ipynb`: Complete Jupyter Notebook with code and visual bar charts.
- `Final_Trading_Sentiment_Analysis.csv`: Merged final clean dataset.
